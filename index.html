Web-технологии для разработчиков

Современные веб-технологии предоставляют разработчикам неограниченные возможности для реализации своих идей. Для того, чтобы вы могли использовать весь их потенциал, необходимо знать как правильно их использовать. На этой странице мы постарались собрать ссылки на документацию по различным технологиям, которые могут быть вам полезны.

Основы веб разработки

HTML

Язык гипертекстовой разметки — это язык, используемый для определения структуры и описания содержания веб-страницы в структурированной форме.

CSS

Каскадные таблицы стилей используются для описания внешнего вида веб-контента.

HTTP

HyperText Transfer Protocol (HTTP) — это протокол, по которому доставляются HTML и прочие медиа-документы.

JavaScript

Язык программирования, широко используемый для реализации взаимодействия пользователя с веб-сайтами и приложениями.

JSON

Текстовый формат данных, следующий за синтаксисом объекта JavaScript

NodeJS

NodeJS - это обычный JavaScript, но работающий не в браузере (на клиенте), а на сервере

01 Веб Компоненты

Веб Компоненты - это набор различных технологий, позволяющих создавать собственные переиспользуемые элементы - со своей функциональностью, инкапсулированной от остального кода - и использовать их в ваших веб-приложениях.

02Web Performance

Производительность в web - это объективные измерения и пользовательские ощущения, связанные с загрузкой и работой приложения. Производительность - это о том, как долго сайт грузится, становится интерактивным и отзывчивым, о том, как плавно происходит взаимодействие с контентом. Плавный ли скролл страницы? Все ли кнопки кликабельны? Всплывающие окна загружаются и показываются быстро? А анимируются? Хорошая производительность требует учёта всех аспектов: как объективных, например, фактическое время загрузки страницы или частота смены кадров; так и субъективных - в буквальном смысле "как пользователь воспринимает систему".

03 Attribution Reporting API

API атрибуции отчетов позволяет разработчикам измерять конверсии — например, когда пользователь нажимает на рекламу, размещенную на одном сайте, а затем переходит на сайт продавца, чтобы совершить покупку, — и получать доступ к отчетам об этих конверсиях. 04 Navigation Sources

Источники навигации, которые заставляют браузер сохранять исходные данные в ответ на навигацию — например, когда пользователь нажимает на ссылку или активирует ее с помощью клавиатуры, или когда навигация происходит в результате Window.open()вызова.

05 Structuring content with HTML

HTML — это технология, которая определяет содержимое и структуру любого веб-сайта. При правильном написании она также должна определять семантику (смысл) содержимого в машиночитаемом виде, что крайне важно для обеспечения доступности, поисковой оптимизации и использования встроенных функций браузеров для оптимальной работы с контентом. В этом модуле рассматриваются основы языка, а также ключевые области, такие как структура документа, ссылки, списки, изображения, формы и многое другое.
